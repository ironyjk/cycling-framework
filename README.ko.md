# cycling-framework

사이클링 코칭 메타 라우터 — 파워존(Coggan), 주기화(Friel), 양극화 훈련(Seiler), 영양 보급, 바이크 핏, 실내 훈련, 장거리 이벤트, 부상 예방 8개 프레임워크.

## 호출

```
/ride
```

## 프레임워크

| 스킬 | 내용 |
|------|------|
| `power-zones` | FTP 기반 존 시스템 (Coggan 7존) |
| `periodization` | 연간 훈련 계획 (Friel/TSS 기반) |
| `polarized-training` | 80/20 강도 분배 (Seiler) |
| `nutrition-fueling` | 레이스·훈련 영양 전략 |
| `bike-fit-equipment` | 핏 프로토콜·장비 선택 |
| `indoor-training` | 트레이너 워크아웃·구조적 세션 |
| `endurance-events` | 그란폰도·스테이지 레이스 준비 |
| `injury-prevention` | 사이클링 부상 예방 |

## 필요 환경

- Claude Code (Skill 툴 활성화)
- [claude-frameworks-marketplace](https://github.com/ironyjk/claude-frameworks-marketplace) 설치
