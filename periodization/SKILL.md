# 사이클링 주기화 (Periodization — Joe Friel)

## 한줄 요약

**목표 이벤트 날짜에서 역산하여 Prep→Base→Build→Peak→Race→Transition 단계를 배치하고, 각 단계에서 볼륨과 강도의 비중을 체계적으로 조절한다.** "언제 뭘 해야 하는가"의 설계도.

## 근거 강도

- 주기화 원론 (Bompa, Matveyev): **강함** (수십 년간 올림픽·프로 스포츠에서 검증)
- Friel 사이클링 특화 ATP: **중간~강함** (실무 표준이지만 개인 반응 차이 큼)
- 블록 주기화 vs 전통 주기화: **중간** (Issurin, Rønnestad — 상급자에겐 블록이 효과적일 수 있음)

## 연간 훈련 계획 (ATP) 단계

### 1. Transition (전환기) — 2~4주
- 시즌 종료 후 심신 회복
- 구조화된 훈련 없음. 자유롭게 타거나 다른 활동
- 볼륨: 시즌 중 30~50%
- 강도: Z1~Z2만
- **목적**: 번아웃 방지, 동기 재충전

### 2. Preparation (준비기) — 2~4주
- 다시 구조화된 훈련 시작
- 근력 훈련 비중 높음 (체육관)
- 라이딩: Z2 위주, 짧은 세션
- **목적**: 근골격계 준비, 코어·근력 기초

### 3. Base 1~3 (기초기) — 8~12주
- 사이클링 훈련의 **핵심 투자 기간**
- **Base 1** (3~4주): Z2 볼륨 점진 증가. 근력 훈련 병행
- **Base 2** (3~4주): Z2 볼륨 최대. 템포(Z3) 도입
- **Base 3** (3~4주): Z2 유지. Z4(역치) 인터벌 도입. 근력 유지로 전환
- 볼륨 증가 규칙: **주당 최대 10% 증가**, 3주 증가 → 1주 회복(Recovery Week)
- **목적**: 유산소 엔진 구축, LT1 끌어올리기

### 4. Build 1~2 (강화기) — 4~8주
- 레이스 특이적 강도 도입
- **Build 1** (3~4주): Z4 역치 인터벌 주 2회. 볼륨 약간 감소
- **Build 2** (3~4주): Z5 VO2max 인터벌 도입. 레이스 시뮬레이션
- 3주 로드 → 1주 회복 유지
- **목적**: FTP 직접 향상, 레이스 강도 적응

### 5. Peak (피킹기) — 1~2주
- 볼륨 대폭 감소 (Build의 40~60%)
- 고강도 짧은 세션 유지 (신경근 자극)
- TSB를 +10~+25로 끌어올림
- **목적**: 피로 해소하면서 체력 유지 → 레이스 당일 최고 컨디션

### 6. Race (경기기)
- 이벤트 주: 가벼운 오프너 세션 (레이스 2일 전)
- 이벤트 후 1~3일 회복
- 시즌 중 여러 이벤트가 있으면 Build~Peak~Race를 반복

## 주간 구조 (미시사이클)

### Base 기간 예시 (주 8시간)
```
월: 휴식 또는 근력 훈련
화: Z2 90min
수: Z2 60min + 근력
목: Z2 90min (또는 Base 3에서 Z4 인터벌)
금: 휴식
토: Long Ride Z2 3~4h
일: Z2 60~90min
```

### Build 기간 예시 (주 8시간)
```
월: 휴식
화: Z4 인터벌 (3×12min @Z4 / 5min rest) 총 75min
수: Z2 60min (회복)
목: Z5 인터벌 (5×4min @Z5 / 3min rest) 총 70min
금: 휴식
토: Long Ride Z2 2.5~3h (마지막 30min Z3)
일: Z2 60min
```

### Recovery Week (4주차)
- 볼륨: 직전 주의 **50~60%**
- 강도: Z4/Z5 세션 → Z2로 대체
- 완전 휴식일 추가
- **건너뛰지 말 것** — 적응은 회복 중에 일어남

## 이벤트 역산 계획 예시

### 16주 그란폰도 준비 (목표: 180km 완주)
```
Week 1~4:   Base 1 (Z2 볼륨 구축)
Week 5~8:   Base 2~3 (Z2 최대 + 템포 도입)
Week 9~12:  Build 1 (Z4 인터벌)
Week 13~14: Build 2 (Z5 인터벌 + 이벤트 시뮬레이션)
Week 15:    Peak (볼륨 50% 감소)
Week 16:    Race Week (오프너 + 이벤트)
```

### 12주 FTP 향상 (목표: +15~20W)
```
Week 1~4:   Base 2~3 (Z2 기반 + 템포)
Week 5~8:   Build 1 (Z4 2×/주)
Week 9~11:  Build 2 (Z4+Z5 혼합)
Week 12:    FTP 재테스트
```

## 한국 사이클리스트 현실

- **겨울 11~2월**: Base 기간과 겹침. 실내 트레이너 필수. 근력 훈련 병행 최적 시기
- **봄 3~4월**: 날씨 좋아지면 야외 복귀. 급격한 볼륨 증가 주의 (10% 규칙)
- **여름 6~8월**: 폭염으로 야외 훈련 제한. 이른 아침/실내 혼합
- **가을 9~10월**: 대부분의 그란폰도/대회 시즌. Peak 타이밍
- **직장인 시간 제약**: 주 6~8시간이 현실적 상한. 질(강도 배분) > 양
- **멀티 이벤트**: 시즌 중 3~5개 이벤트 → Build~Peak~Race 미니사이클 반복

## 안티패턴

- **Base 건너뛰고 바로 인터벌** — "집 지을 때 기초 없이 2층부터" 와 같음
- **Recovery Week 생략** — 3주 로드 후 1주 회복은 선택이 아닌 필수
- **매주 같은 훈련 반복** — 주기화 없이 1년 내내 같은 패턴 = 정체
- **Peak 없이 레이스** — 피로가 쌓인 채로 이벤트 → 성적 부진
- **볼륨 급증** — 주당 10% 초과 증가는 부상 위험 급상승
- **여러 목표 동시 추구** — "FTP도 올리고 체중도 줄이고 근육도 키우기"는 불가능. 시기별 우선순위

## 한계

1. 아마추어는 직장·가정으로 인해 계획대로 진행이 어려움 → 유연한 조정 필요
2. 개인 반응 차이 — 3주 로드/1주 회복이 안 맞는 사람도 있음 (2:1로 조정)
3. 블록 주기화가 상급자에겐 더 효과적일 수 있음
4. Base 기간의 "지루함"을 견디기 어려움 → 소셜 라이딩, 경치 좋은 코스로 보상
