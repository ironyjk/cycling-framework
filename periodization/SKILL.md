# Cycling Periodization (Periodization — Joe Friel)

> **Background and theory**: Read [references/foundation.md](references/foundation.md)


## One-Line Summary

**Work backward from target event dates to arrange Prep→Base→Build→Peak→Race→Transition phases, systematically adjusting the balance of volume and intensity at each stage.** The blueprint for "what to do and when."


## Weekly Structure (Microcycle)

### Base Period Example (8 hours/week)
```
Mon: Rest or strength training
Tue: Z2 90min
Wed: Z2 60min + strength
Thu: Z2 90min (or Z4 intervals in Base 3)
Fri: Rest
Sat: Long Ride Z2 3–4h
Sun: Z2 60–90min
```

### Build Period Example (8 hours/week)
```
Mon: Rest
Tue: Z4 intervals (3×12min @Z4 / 5min rest) total 75min
Wed: Z2 60min (recovery)
Thu: Z5 intervals (5×4min @Z5 / 3min rest) total 70min
Fri: Rest
Sat: Long Ride Z2 2.5–3h (last 30min Z3)
Sun: Z2 60min
```

### Recovery Week (Week 4)
- Volume: **50–60%** of the preceding week
- Intensity: Replace Z4/Z5 sessions with Z2
- Add additional full rest days
- **Do not skip** — adaptation occurs during recovery


## Event Reverse-Planning Examples

### 16-Week Gran Fondo Preparation (Goal: 180km completion)
```
Week 1–4:   Base 1 (Z2 volume building)
Week 5–8:   Base 2–3 (Z2 maximum + tempo introduction)
Week 9–12:  Build 1 (Z4 intervals)
Week 13–14: Build 2 (Z5 intervals + event simulation)
Week 15:    Peak (50% volume reduction)
Week 16:    Race Week (opener + event)
```

### 12-Week FTP Improvement (Goal: +15–20W)
```
Week 1–4:   Base 2–3 (Z2 foundation + tempo)
Week 5–8:   Build 1 (Z4 2×/week)
Week 9–11:  Build 2 (Z4 + Z5 mixed)
Week 12:    FTP retest
```


## Practical Considerations for Korean Cyclists

- **Winter (Nov–Feb)**: Overlaps with Base period. Indoor trainer essential. Optimal time for concurrent strength training
- **Spring (Mar–Apr)**: Return outdoors as weather improves. Beware of abrupt volume increases (10% rule)
- **Summer (Jun–Aug)**: Extreme heat limits outdoor training. Mix early morning rides with indoor sessions
- **Autumn (Sep–Oct)**: Primary gran fondo and race season. Peak timing
- **Working professional time constraints**: 6–8 hours/week is a realistic ceiling. Quality (intensity distribution) > quantity
- **Multi-event seasons**: 3–5 events per season → repeat Build→Peak→Race mini-cycles


## Anti-Patterns

- **Skipping Base and jumping straight to intervals** — Like building the second floor without laying a foundation
- **Omitting Recovery Weeks** — 3 weeks load followed by 1 week recovery is mandatory, not optional
- **Repeating the same training every week** — No periodization, same pattern year-round = plateau
- **Racing without a Peak phase** — Entering an event with accumulated fatigue → poor performance
- **Sudden volume spikes** — Exceeding 10% weekly increase sharply raises injury risk
- **Pursuing multiple goals simultaneously** — "Raise FTP, lose weight, and build muscle all at once" is not feasible. Prioritize by phase


## Limitations

1. Amateurs face work and family constraints that make strict plan adherence difficult → flexible adjustment required
2. Individual response varies — the 3:1 load-to-recovery ratio does not suit everyone (adjust to 2:1 if needed)
3. Block periodization may be more effective for advanced athletes
4. The "monotony" of Base period is hard to endure → compensate with social rides and scenic routes
