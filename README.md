# cycling-framework

Cycling coaching meta-router — 8 evidence-based frameworks covering Power Zones (Coggan), Periodization (Friel), Polarized Training (Seiler), Nutrition & Fueling, Bike Fit & Equipment, Indoor Training, Endurance Events, and Injury Prevention.

## Invoke

```
/ride
```

## Frameworks

| Skill | Focus |
|-------|-------|
| `power-zones` | FTP-based zone system (Coggan 7-zone) |
| `periodization` | Annual training plan (Friel/TSS-based) |
| `polarized-training` | 80/20 intensity distribution (Seiler) |
| `nutrition-fueling` | Race and training nutrition |
| `bike-fit-equipment` | Fit protocol and gear selection |
| `indoor-training` | Trainer workouts and structured sessions |
| `endurance-events` | Gran Fondo, sportive, and stage race prep |
| `injury-prevention` | Common cycling injuries and prevention |

## Requirements

- Claude Code with Skill tool enabled
- Installed via [claude-frameworks-marketplace](https://github.com/ironyjk/claude-frameworks-marketplace)
