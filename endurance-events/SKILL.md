# 장거리 이벤트 준비 (Endurance Events)

## 한줄 요약

**목표 이벤트의 거리·고도·특성을 분석하고, 12~16주 준비 계획을 세운다.** 페이싱 전략, 보급 계획, 클라이밍 기술이 완주와 DNF를 가른다.

## 근거 강도

- 주기화 기반 이벤트 준비: **강함**
- 네거티브 스플릿 페이싱: **강함** (전반 보수적 → 후반 유지/가속이 최적)
- 드래프팅 효과: **강함** (25~40% 파워 절약, 속도·풍향에 따라)
- 클라이밍 좌석 vs 댄싱: **중간** (개인 효율 차이, 일반적으로 좌석이 효율적)

## 이벤트 유형별 특성

| 유형 | 거리 | 특징 | 핵심 능력 |
|---|---|---|---|
| 센추리 | 160km | 장거리 기본 | Z2 지구력, 보급 |
| 그란폰도 | 100~200km | 언덕 포함, 커트라인 있을 수 있음 | 클라이밍 + 지구력 |
| 브레베 (200km) | 200km | 자율 주행, 체크포인트 | 자기 관리, 야간 주행 |
| 브레베 (300~600km) | 300~600km | 수면/보급 자율 관리 | 극한 지구력, 멘탈 |
| 타임트라이얼 | 20~40km | 개인 전력 | FTP, 에어로 |

## 12~16주 준비 계획 (그란폰도 180km 예시)

### 전제조건
- 현재 주 5~8시간 훈련 중
- 최장 라이딩 80~100km
- FTP 알고 있음

### 주차별 계획
```
Phase 1: Base 확장 (Week 1~4)
- 주말 Long Ride를 매주 10~15km씩 증가
- 100km → 115km → 130km → 회복주 80km
- 평일: Z2 60~90min × 2~3회
- 주 TSS: 350~450

Phase 2: 강도 도입 (Week 5~8)
- Long Ride: 140km → 150km → 160km → 회복주 100km
- Z4 인터벌 주 1회 도입 (3×12min @Z4)
- 이벤트 코스와 유사한 지형에서 훈련
- 주 TSS: 450~550

Phase 3: 이벤트 특이적 (Week 9~12)
- Long Ride에 레이스 페이스 구간 포함
- 클라이밍 반복 (실제 코스 고도와 유사)
- 보급 전략 실전 테스트
- VO2max 인터벌 주 1회
- 주 TSS: 500~600

Phase 4: 테이퍼 + 레이스 (Week 13~16)
- Week 13: 마지막 Long Ride (140km, 편안한 페이스)
- Week 14: 볼륨 60%, 짧은 고강도 유지
- Week 15: 볼륨 40%, 오프너 세션
- Week 16: 이벤트 주 (2일 전 30min 오프너)
```

## 페이싱 전략

### 네거티브 스플릿 원칙
- **전반 50%**: 목표 평균의 90~95%로 보수적
- **후반 50%**: 에너지 남아있으면 점진 가속
- **이유**: 글리코겐 고갈은 비가역적. 초반 과페이스 = 후반 붕괴

### 파워 기반 페이싱 (파워미터 있을 때)
| 이벤트 시간 | 목표 NP | 비고 |
|---|---|---|
| 3~4시간 | FTP의 70~80% | 센추리 |
| 5~6시간 | FTP의 65~75% | 그란폰도 |
| 8시간+ | FTP의 55~65% | 브레베 200km |
| 12시간+ | FTP의 45~55% | 브레베 300km+ |

### 심박 기반 (파워미터 없을 때)
- 최대 심박의 65~75% 유지
- 언덕에서 최대 80%까지 허용, 내리막에서 회복
- 심박 드리프트 주의 (3시간 후 같은 파워에서 HR 10% 상승)

## 클라이밍 전략

### 기본 원칙
- **일정 파워 유지** — 언덕 초입에서 가속하지 말 것
- 평지 대비 NP 10~15% 높은 것은 괜찮음 (짧은 구간 + 내리막 회복)
- 긴 언덕 (20min+): FTP의 85~95%
- 짧은 언덕 (5min): FTP의 100~110% 가능

### 좌석(시팅) vs 댄싱(스탠딩)
| 항목 | 시팅 | 댄싱 |
|---|---|---|
| 효율 | 높음 (심박 낮음) | 낮음 (HR 5~10bpm 높음) |
| 용도 | 긴 완만한 언덕 | 짧은 급경사, 가속 |
| 근육 | 대퇴사두 중심 | 전신 (체중 활용) |
| 추천 | 기본 모드 (80%) | 경사 10%+ 또는 리듬 변화 (20%) |

### 케이던스
- 평지: 85~95rpm (개인 선호)
- 언덕: 70~85rpm (저케이던스 고토크)
- 가파른 경사 (10%+): 60~75rpm (기어 한계)
- 너무 낮은 케이던스 (<60) → 무릎 부담 증가

### 하강 전략
- **안전 최우선** — 기록은 오르막에서 만든다
- 코너 진입 전 충분한 감속
- 바깥→안쪽→바깥 라인 (하지만 맞은편 차량 주의)
- 브레이크: 양손 교대 (손가락 피로 방지)
- 하강은 회복 구간 — 페달링하면서 심박 낮추기

## 그룹 라이딩

### 드래프트 효과
- 앞 라이더 뒤 30~50cm: **25~40% 파워 절약**
- 두 번째 라이더도 3~5% 절약
- 측풍 시: 바람 반대쪽으로 약간 빗겨서 위치

### 로테이션
- 앞 라이더: 30초~2분 후 교대 (체력/인원에 따라)
- 교대 시: 부드럽게 옆으로 빠지고 뒤로
- **당기지 않아도 됨** — 힘들면 뒤에서 빠르게 교대

### 안전 수신호
- 오른손/왼손 뒤로 내밀기 → 장애물 방향
- 손 흔들기 → 감속
- 가리키기 → 장애물, 구멍
- 소리: "차!", "앞!", "구멍!"

## 한국 주요 이벤트

| 이벤트 | 거리 | 고도 | 시기 | 특징 |
|---|---|---|---|---|
| 서울-부산 종주 | 525km | ~3,000m | 연중 | 4대강 자전거길, 2~4일 |
| 제주 환상 자전거길 | 180km (해안) / 234km (전체) | ~1,500m | 봄·가을 | 바람 주의, 경치 최고 |
| 영주 소백산 GF | 120~150km | ~2,500m | 가을 | 언덕 중심 |
| 새만금 라이딩 | 80~150km | 평탄 | 여름 | 평지, 바람 |
| 브레베 시리즈 | 200/300/400/600km | 다양 | 3~10월 | 한국 랜도뇌르 |
| 각 지역 그란폰도 | 80~200km | 다양 | 봄·가을 | 지역 축제 연계 |

### 4대강 종주 팁
- **인증센터**: 자전거 여권 스탬프 필수
- **숙소**: 자전거 도로 근처 모텔/펜션 예약 (성수기 매진)
- **보급**: 편의점 간격 20~40km. 여분 보급 필수
- **안전**: 자전거 전용 도로지만 보행자·전동 킥보드 주의

## 안티패턴

- **첫 이벤트에서 기록 노리기** — 첫 이벤트는 완주가 목표. 기록은 두 번째부터
- **훈련에서 이벤트 거리를 타지 않기** — 최소 이벤트 거리의 75~80%는 훈련에서 경험 필요
- **새로운 장비/보급으로 이벤트 출전** — 검증되지 않은 것은 모두 위험
- **집단에서 무리하게 따라가기** — 자기 페이스 벗어나면 후반 붕괴
- **하강에서 속도 경쟁** — 가장 위험한 구간. 안전 > 속도

## 한계

1. 날씨·바람은 예측 불가 — 계획 B 필수 (커트라인 조절, 보급 추가)
2. 그룹 역학은 예측 어려움 — 너무 빠르거나 느린 그룹과 합류 시 판단
3. 극한 이벤트(브레베 600km+)는 수면·정신력이 핵심 — 체력 이상의 영역
